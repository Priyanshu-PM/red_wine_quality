# red-wine-quality

## Installation

1. Clone the repository:

```shell
git clone https://github.com/somesh4545/red-wine-quality.git
```

2. Change dir to folder cloned:
```shell
cd folder
pip install -r requirements.txt
```

3. Configuration
```shell
set FLASK_APP=app.py
flask run
```

You will find the project running on 
http://localhost:5000
